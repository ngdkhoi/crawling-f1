How to use this

### Require
 - node 16.x
 - psql
### Install nestjs
  run `npm install -g @nestjs/cli`
### create database
 - run `db:dev:restart` to build/re-build dockercompose file

### run project
 - run `yarn start:dev` to run develop
 - open  `http://loalhost:3000/api` to test api
 - run `npx prisma studio` to view database
